## Ex002
#### A palindrome is a sequence of characters that reads the same backwards and forwards. Given a string, s, find the longest palindromic substring in s.

##### Examples:
```
Input: "banana"
Output: "anana"
```
```
Input: "million"
Output: "illi"
```
